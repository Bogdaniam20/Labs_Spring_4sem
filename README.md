Для проверки использую Postman
Для отправки запроса выбираю метод POST
URL: http://localhost:8080/parse
На вкладке Body выбираю raw и ввожу Json в виде 
{
    "text": "Мой email: example@example.com и телефон +1234567890."
}
Сервис должен вернуть Json с найденными email-адресами и телефонами. Пример ответа:
{
    "emails": ["example@example.com"],
    "phones": ["+1234567890"]
}
